题目编号：67

题目：二进制求和

难度：简单

考察范围：字符串、数学

题干：

给定两个二进制字符串，返回他们的和（用二进制表示）。

输入为非空字符串且只包含数字 1 和 0。

示例 1:

输入: a = "11", b = "1"
输出: "100"

示例 2:

输入: a = "1010", b = "1011"
输出: "10101"

解题思路：

将两个二进制字符串转换为十进制数，相加后再转换为二进制字符串即可。

解决方案：

```python
class Solution:
    def addBinary(self, a: str, b: str) -> str:
        # 将二进制字符串转换为十进制数
        a_int = int(a, 2)
        b_int = int(b, 2)
        # 十进制数相加
        sum_int = a_int + b_int
        # 将十进制数转换为二进制字符串
        sum_str = bin(sum_int)[2:]
        return sum_str
```

算法复杂度：时间复杂度为 O(n)，其中 n 为二进制字符串的长度。
